PyWPS 3.0.0 
2008-09-16
-----------
* Support for OGC(R) WPS 1.0.0

* New code strucuture
    - completely rewritten source base
    - more object-orientation
    - simplification
    - python-htmltml templating system
    - support for internationalization

* New configuration file /etc/pywps.cfg 

* New WPSProcess class with predefined methods for easier process coding
    - addBBoxInput
    - addBBoxOutput
    - addComplexInput
    - addComplexOutput
    - addLiteralInput
    - addLiteralOutput
    - cmd
    - getInput
    - getInputValue
    - i18n
    - message
    - setOutputValue

* New Status class with predefined methods
    - set (msg='', percentDone=0)

* New Input and Output classes
    - setValue
    - getValue
