pywps-4
=======

New version of PyWPS - written from scratch

Dependencies
============

lxml - http://lxml.de
owslib - https://github.com/geopython/OWSLib

Run tests
=========

Python 3
--------
    python3 tests/__init__.py

or

    python3 tests/parse.py
    
Python 2(.7)
------------
    python2 tests/__init__.py

or

    python2 tests/parse.py
